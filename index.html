YAK Real-time Operating System

A simple, fully-functional, real-time 8086 kernel.

In order to better understand the challenges associated with embedded and real-time systems, a classmate and I implemented a simple real-time kernel called YAK.

This simple kernel was written in C and 8086 assembly to allow preemptive context switching, semaphores, message queues, and events. Application code could utilize multiple tasks and use all of the functionality of our simple RTOS.

We applied our knowledge of the hardware/software interface to create our RTOS, and designed a real-time application using it.

We also gained experience with interrupt service routines from both the kernel development and application development perspectives.

Check out the YAK RTOS on GitHub.
